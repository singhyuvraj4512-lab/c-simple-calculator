# 🔢 C Simple Calculator

A menu-driven CLI-based calculator written in C that performs common arithmetic operations, including additional operations like modulus and power. The program includes basic error handling for invalid input and division/modulus by zero.

---

## 🚀 Features

- Addition, subtraction, multiplication, and division  
- Modulus operation (integer-based)  
- Power calculation using math library  
- Menu-driven interface with continuous execution  
- Error handling for invalid input and division/modulus by zero  

---

---

## ⚙️ How to Run

1. Compile the program:

```bash
gcc calculator.c -o calculator -lm
```

2. Run the program:

```bash
./calculator
```
